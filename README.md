# Análisis de Valor de Jugadores - Liga MX

## Descripción
Proyecto de análisis de datos deportivos que evalúa el rendimiento y eficiencia de jugadores de la Liga MX usando estadísticas reales de FBref.

## Objetivo
Identificar jugadores sobrevalorados e infravalorados mediante métricas de rendimiento propias, combinando análisis financiero con datos deportivos.

## Métricas desarrolladas
- **Contribuciones**: Goles + Asistencias por jugador
- **Goles por partido**: Eficiencia goleadora
- **Índice de Valor**: Contribuciones por cada 100 partidos jugados

## Tecnologías utilizadas
- Python 3.14
- Pandas
- Matplotlib
- Seaborn

## Fuente de datos
FBref - Liga MX Player Standard Stats (actualizado por jornada)
